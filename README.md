# MineCLI - A Simple, Minimal Way to Enjoy the beauty of Classic Minecraft Versions Locally

MineCLI is a straightforward solution for playing old Minecraft versions locally, without the need for _Microslop_'s DRM or account-based login.

The project is composed of two main parts:
- **Installer**: A tool that provides an easy and fast way to set up both a Minecraft server and client on your system.
- **Launcher**: It simply launches a cracked version of Prism Launcher made by [Diegiwg](https://github.com/Diegiwg).

> Note that this is, as of now, only meant to work for apt package manager based Linux systems.\
> I might consider including other package managers in the near future such as pacman and portage.
