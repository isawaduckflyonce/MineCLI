MineCLI is a simple and minimal way to enjoy the beauty of old Minecraft versions locally, without ever dealing with any Microslop DRM or such.

The project is divided in a simple installer and a launcher. The installer is meant to provide a fast way of setting both a server and a client up on the system, which can then be player with the launcher.

Note that this is only meant to work for apt package manager based linux systems, I might consider including other package managers in the near future such as pacman and portage.
